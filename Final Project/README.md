# School Chatbot Project

This project is a simple school inquiry chatbot system with three main components:

- **Frontend**: A web-based chatbot interface built with Vite (JavaScript/HTML/CSS).
- **Spring Boot Backend**: Acts as an API gateway, forwarding requests from the frontend to the AI service.
- **AI Service (Python/FastAPI)**: Handles natural language questions using an LLM (Groq API) and returns professional responses.

---

## Features
- Chatbot UI for parents/guardians to ask school-related questions.
- Professional, polite, and clear responses generated by an LLM.
- Easily extensible for more categories or advanced logic.

## Setup & Run

### 1. AI Service (Python/FastAPI)

- Install dependencies:
  ```sh
  pip install fastapi uvicorn groq python-dotenv
  ```
- Set your Groq API key in a `.env` file:
  ```env
  GROQ_API_KEY=your_groq_api_key_here
  ```
- Run the service:
  ```sh
  uvicorn main:app --reload --port 8000
  ```

### 2. Spring Boot Backend

- Open the `springboot_app` folder in your IDE.
- Make sure you have Java and Maven installed.
- Build and run:
  ```sh
  mvn spring-boot:run
  ```
- The backend should run on `http://localhost:8081` and forward `/api/ask` requests to the AI service.

### 3. Frontend (Vite)

- Open the `final-project` folder.
- Install dependencies:
  ```sh
  npm install
  ```
- Start the dev server:
  ```sh
  npm run dev
  ```
- Open the browser at the URL shown (usually `http://localhost:5173`).

---

## How it Works

1. User enters a question in the chatbot UI.
2. Frontend sends the question to the Spring Boot backend (`/api/ask`).
3. Spring Boot forwards the question to the FastAPI AI service (`/ask`).
4. The AI service generates a response using the Groq LLM and returns it.
5. The answer is displayed in the chatbot UI.
